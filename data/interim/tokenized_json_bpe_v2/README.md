# Estructura de `data/interim/tokenized_json_bpe_v2`

Esta carpeta contiene la segunda version de los JSON tokenizados para preentrenamiento.

Estructura prevista:

```text
data/interim/tokenized_json_bpe_v2/
├── maestro-v3.0.0/
│   ├── 2004/
│   ├── 2006/
│   └── ...
└── ariamidi/
    ├── aa/
    ├── ab/
    └── ...
```

La estructura mantiene la procedencia del corpus. En ARIAMidi se documenta el patron con ejemplos de carpetas, ya que enumerarlas todas no aporta informacion adicional.
