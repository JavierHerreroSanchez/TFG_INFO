# Estructura de `data/interim/tokenized_json_bpe`

Esta carpeta contiene la primera version de los JSON tokenizados con REMI+BPE para preentrenamiento.

Estructura prevista:

```text
data/interim/tokenized_json_bpe/
├── maestro-v3.0.0/
│   ├── 2004/
│   ├── 2006/
│   └── ...
└── ariamidi/
    ├── aa/
    ├── ab/
    └── ...
```

La estructura reproduce la organizacion del corpus de origen. En ARIAMidi se indican solo ejemplos de subcarpetas, no el listado completo.
