# Estructura de `data/interim/tokenized_json_bpe`

Primera tokenizacion REMI+BPE usada para preentrenamiento.

```text
data/interim/tokenized_json_bpe/
|-- maestro-v3.0.0/
|   |-- 2004/
|   |-- 2006/
|   `-- ...
`-- ariamidi/
    |-- aa/
    |-- ab/
    `-- ...
```

La estructura replica la procedencia de los MIDIs originales.
